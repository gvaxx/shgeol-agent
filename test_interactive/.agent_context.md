# Repo Context

## Purpose
This repository provides a simple ledger system to track financial transactions, including income and expenses. It calculates balances, generates summaries, and supports reporting features. The core functionality is extended by utility functions for parsing and formatting monetary values.

## Architecture Notes
- **ledger.py**: Core module containing the `Ledger` class for managing transactions, calculating balances, and generating summaries.
- **report.py**: Dependent module that uses `Ledger` to produce human-readable reports and demonstrations.
- **utils.py**: Utility module with helper functions for parsing/formating amounts and general value clamping.
- Modules follow a separation of concerns pattern, with `ledger.py` handling data logic, `report.py` managing output, and `utils.py` providing reusable helpers.
